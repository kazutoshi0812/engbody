# Weblab_coffee-life

## 作業ディレクトリ
- sass/jsの記述はsrcフォルダの中で行う
- 画像はsrcフォルダのimagesの中に格納する
- コンパイルされたCSS/jsと圧縮された画像はdistフォルダの中に出力される
- htmlはdist直下のhtmlファイルに直接記述する
